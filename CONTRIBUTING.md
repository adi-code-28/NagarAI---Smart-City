# Contributing to NagarAI

We're excited that you're interested in contributing to **NagarAI**! As a smart city project, we believe in the power of community-driven innovation.

## 🚀 How to Contribute

1.  **Fork the Repository**: Create your own copy of the project.
2.  **Create a Branch**: Work on your feature or bug fix in a dedicated branch.
3.  **Commit Your Changes**: Write clear, descriptive commit messages.
4.  **Push to Your Fork**: Upload your changes to your GitHub account.
5.  **Submit a Pull Request**: Explain your changes and why they should be merged.

## 🛠️ Development Setup

1.  Install Node.js (v18+).
2.  Run `npm install` to set up dependencies.
3.  Run `npm run dev` to start the local development server.

## 📝 Code Style

-   Use **TypeScript** for all new code.
-   Follow **Tailwind CSS** utility-first principles for styling.
-   Ensure all components are **accessible** and **responsive**.
-   Write **clean, modular code** with helpful comments.

## 🐞 Reporting Bugs

If you find a bug, please open an issue with:
-   A clear description of the problem.
-   Steps to reproduce the issue.
-   Expected vs. actual behavior.

## 💡 Feature Requests

Have an idea to make Delhi smarter? Open an issue and label it as a `feature request`.
