# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of **NagarAI** seriously. If you find a vulnerability, please report it to us by:
1.  Opening a **Security Issue** on GitHub.
2.  Describing the vulnerability and how to reproduce it.
3.  Providing any potential fixes or mitigations.

We will review all reports and take appropriate action to ensure the safety of our smart city platform.
